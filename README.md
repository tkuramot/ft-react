# ft-react

## setup

1. **Clone the repository** (if you haven't already):

```sh
git clone https://github.com/tkuramot/ft_react.git
cd ft_react
```

2. **Install dependancies**:

```sh
pnpm i
```

3. **Start the development server**:

```sh
make d
```
